## macGPT

> 第一个版本已经发布，大家可以去release下载！


macOS原生 Application

因为自己是程序员，使用chatGPT web实在是不方便，需要不断刷新。所以一直想开发一个chatGPT mac客户端。

偶然看见https://github.com/LeaderBoy/Chat

但是该项目不是我想象中的样子，所以拿此项目进行改造。



目前问题还有很多，下面是已经开发完了的：

- 多会话聊天
- 会话保存

当然还有很多还没做，例如管理apikey、新增会话...

我只是先把代码推上来，后面会继续完善。
